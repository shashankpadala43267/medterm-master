MedTerm Question Importer

Upload a CSV or JSON question file. The importer validates it and creates a ready-to-upload questions.json file.

This tool runs entirely in your browser. It does not upload the question file anywhere.

Required CSV columns

question, choice1, choice2, choice3, choice4, answer, category, explanation

The answer value may be 1–4, 0–3, or A–D.
